# 🛡️ Plataforma Avanzada de Auditoría Forense y Mitigación de Riesgos — Aseguradora del Sur

Solución analítica híbrida de nivel empresarial diseñada para la automatización inteligente, detección de anomalías y auditoría de control operativo en el procesamiento de siniestros.

## 🔬 Pilares Tecnológicos del Núcleo Analítico
* **Motor de Evaluación Determinista:** Validaciones cruzadas de alta velocidad sobre el ecosistema de datos.
* **Modelos de Anomalías Estadísticas:** Identificación de casos atípicos mediante algoritmos de aislamiento matemático.
* **Procesamiento Cognitivo Multimodal:** Extracción semántica y peritaje documental mediante modelos fundacionales de lenguaje.
* **Subsistema de Inspección Visual:** Auditoría de consistencia estilométrica y trazos morfológicos en evidencias manuscritas.

## 🚀 Requisitos de Ejecución
1. Instalar dependencias: `pip install -r requirements.txt`
2. Configurar variables de entorno locales basándose en la plantilla anónima de referencia.
3. Inicializar el gateway centralizado: `python main.py`
